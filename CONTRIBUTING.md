# Contributing

## Setup

1. Install dependencies with `npm install`
2. Start the app with `npm start`
3. Run checks before opening a pull request:
   - `npm run lint`
   - `npm test`

## Pull Requests

- Keep changes focused and scoped
- Describe behavior changes clearly
- Include screenshots for UI changes when relevant
- Prefer Conventional Commits such as `feat:` and `fix:`

## Issues

- Use the issue templates when available
- Include reproduction steps, expected behavior, and current behavior
- For environment-specific problems, include OS and runtime details

## Conduct

By participating in this project, you agree to follow the
[Code of Conduct](./CODE_OF_CONDUCT.md).
